    A. Apa Itu php artisan serve?

    php artisan serve adalah perintah untuk menjalankan:

    Built-in development server Laravel.

    Command:

    php artisan serve


    Biasanya hasilnya:

    Starting Laravel development server: http://127.0.0.1:8000


    Artinya aplikasi kamu sekarang bisa diakses di browser.

    
    
    
    
    
    B. Apa Itu Artisan?

    artisan adalah:

    CLI (Command Line Interface) resmi Laravel.

    Semacam control panel lewat terminal.

    Contoh command lain:

    php artisan route:list
    php artisan make:controller UserController
    php artisan migrate

    J. Mental Model Saat Server Jalan

    Ingat ini bro:

    Laravel itu tidak berjalan terus-menerus.

    Yang berjalan adalah:

    PHP built-in server

    Setiap request:

    PHP menjalankan index.php

    Laravel booting ulang

    Proses request

    Response dikirim

    Selesai

    Laravel itu stateless per request.
